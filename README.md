# kxrs__ profile page

Minimal personal profile page deployed on Vercel.

## Overview

This project is a lightweight personal profile page that displays:

- Discord presence (via Lanyard API)
- GitHub profile information
- Dynamic activity cards
- Terminal-style animated background

The page is designed to be minimal, fast, and visually similar to Discord UI.

## Tech Stack

- HTML / CSS / Vanilla JS
- Serverless Functions
- Lanyard API
- GitHub API

## Features

- Real-time Discord presence
- Spotify activity display
- GitHub profile card
- WebSocket updates
- Lightweight UI with no frameworks
